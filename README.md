# Antigravity - Advanced MERN Stack Portfolio

Antigravity is a next-generation developer portfolio platform built with the MERN stack (MongoDB, Express, React/Next.js, Node.js). It features a modern, animated UI, a fully functional admin dashboard for content management, and a production-ready architecture.

- **Advanced Frontend**: Next.js 14 App Router, TypeScript, Tailwind CSS, Framer Motion.
- **Robust Backend**: Node.js, Express, MongoDB with Mongoose.
- **Secure Authentication**: JWT-based auth for Admin Dashboard.
- **Dynamic Content**: Manage Projects, Blogs, and Messages via Admin Panel.
- **File Uploads**: Cloudinary integration for image management.
- **Modern UI**: Clean, premium aesthetics with smooth animations.
- **SEO Optimized**: Server-Side Rendering (SSR) and semantic HTML.

## üèÅ Getting Started

### Prerequisites

- Node.js (v18+)
- MongoDB Atlas URI
- Cloudinary Account

### Installation

1. **Clone the repository**
   ```bash
   git clone <repo-url>
   cd antigravity
   ```

2. **Setup Backend**
   ```bash
   cd antigravity-backend
   npm install
   # Create .env file with your credentials
   npm run dev
   ```

3. **Setup Frontend**
   ```bash
   cd antigravity-frontend
   npm install
   # Create .env.local file if needed
   npm run dev
   ```

4. **Access the App**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:5000
   - Admin Login: http://localhost:3000/login
